# Shared image library

Map-art source images published from the app land here (committed via the GitHub
Contents API from **Image ▸ Publish current image to repo…**). Workspaces reference an
image by its path in this folder (e.g. `images/castle.png`), so collaborators who open the
same workspace automatically load the identical source image.

Low-volume by design — a handful of images, added once. You can also drop images here
manually and they'll appear in the app's "Shared image library" picker.
